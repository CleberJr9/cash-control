# 💸 Cash Control

Aplicativo mobile de controle de despesas pessoais desenvolvido em Flutter, criado como projeto de estudo com foco em boas práticas de desenvolvimento mobile.

---

## 📱 Sobre o projeto

O Cash Control permite ao usuário acompanhar seus gastos mensais de forma simples e visual. O app exibe um resumo do mês atual, evolução dos gastos por período e lista de transações agrupadas por data.

O projeto está em desenvolvimento ativo e ainda não possui integração com backend — os dados são locais por enquanto.

---

## ✅ Funcionalidades

- Dashboard com total gasto no mês e orçamento restante
- Gráfico de barras com evolução dos gastos por período
- Filtro por período: Diário, Semanal e Mensal
- Lista de despesas agrupada por data
- Tela para adicionar nova despesa com teclado numérico e categoria
- Detalhe de categoria
- Tela de configurações e perfil com toggles
- Navegação com bottom navigation bar e botão de ação flutuante (FAB)

---

## 🛠️ Tecnologias

- **Flutter** — framework principal
- **Dart** — linguagem
- **fl_chart** — gráficos
- **intl** — formatação de moeda brasileira (pt_BR)
- **flutter_svg** — ícones SVG
